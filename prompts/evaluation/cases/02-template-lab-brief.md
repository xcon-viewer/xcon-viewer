# Case 02: Template Lab Business Document

Create a data-bound executive growth brief for a SaaS business.

## Required Visible Syntax

- Markdown
- `xcon-chain-fixture`
- `xcon-chain as ...`
- `xcon-sketch`
- `xcon-workflow` if an approval flow is useful

## Data

The fixture should include:

- revenue
- growth rate
- revenue by channel
- operating plan rows
- approval tasks

## Required Visuals

- Metric hero card
- Bar chart
- SpanGrid operating plan table
- Approval workflow or network diagram

## Must Test

- `$topChannels.` with punctuation should resolve correctly.
- The visual SKETCH block must be complete and renderable.
- The document should still read well as normal Markdown.
