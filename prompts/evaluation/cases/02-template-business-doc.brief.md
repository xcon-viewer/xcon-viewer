# Test Brief: Executive Growth Brief

Create a Markdown document template for an executive growth report.

Requirements:

- Visible syntax must be Markdown + XCON Chain SUGAR + XCON/SKETCH.
- Include one `xcon-chain-fixture` block with realistic business data.
- Include at least four `xcon-chain as alias` blocks.
- Use aliases in normal Markdown text.
- Include one `xcon-sketch` visual brief card.
- Include a chart component, a SpanGrid table, and a workflow approval block if supported by the target prompt.
- Keep SKETCH blocks complete and starting with `screen`.
- Do not use HTML as the primary output format.

Return a complete Markdown document.
