# Test Brief: Mobile Coffee Pickup Home

Create a polished mobile app home screen for a local coffee pickup service.

Requirements:

- Use XCON/SKETCH only.
- The fenced block must be `xcon-sketch`.
- The SKETCH must start with `screen`.
- Screen size: `402x812`.
- Include a header with brand, location, search icon button, and cart icon button.
- Include a hero card with a real visual image from Unsplash.
- Include category chips for Espresso, Latte, Cold Brew, Dessert.
- Include one vertical product list with at least three products.
- The list must include `dataTemplate` and a named `templates.cell` layout.
- Include a fixed bottom navigation with Home, Menu, Orders, Profile.
- Navigation buttons must include `activity` actions where appropriate.
- Icon-only buttons must use supported icon names.

Return only the generated XCON/SKETCH fence.
