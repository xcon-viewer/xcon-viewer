# Public Showcase Fixtures

These fixtures are public viewer examples copied from the compatibility showcase. They are useful for checking full component screens, rich component catalogs, and XCON/SKETCH rendering.

Custom component extension fixtures are intentionally excluded from this public directory.

## Open In Playground

Use SKETCH first for compact authoring and Markdown examples. JSON is the canonical structured form. XML and TAGLESS are included for compatibility and round-trip checks.
